
# 🚀 CHICAS-KELUAR --- IRIC-9-1

### 📋 Descripción del Proyecto
Repositorio oficial del equipo **GIRLS KELUAR - IRIC 9-1**. Esta es una aplicación web mínima desarrollada bajo un entorno multicontenedor con Docker, que incluye la gestión, validación e inicio de sesión de usuarios mediante un CRUD completo integrado con una base de datos MySQL.

---

### 👥 Integrantes del Equipo
* **Arisbethb** (Administradora / DevOps & Infraestructura)
* **keylaam0** (Desarrolladora Fullstack / Frontend & UI)
* **Luma-23** (Desarrolladora Backend / API & Base de Datos)


---

### 🛠️ Arquitectura y Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3, JavaScript y Bootstrap para el diseño responsivo.
* **Backend:** API REST encargada de la lógica del CRUD e inicio de sesión.
* **Base de Datos:** MySQL para el almacenamiento seguro de los usuarios.
* **Orquestación:** Docker & Docker Compose para el despliegue del entorno multicontenedor.

---

### 📦 Estructura del Repositorio
* `/backend` - Código fuente de la API y lógica del servidor.
* `/Interfaz` - Vistas del usuario, Login y componentes del CRUD.
* `docker-compose.yml` - Archivo de configuración para el levantamiento de los contenedores.
